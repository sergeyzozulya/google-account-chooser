Google Account Chooser

Configure which Google services are intercepted and how they open.

General

Enable extension
Temporarily pause all interception without changing service settings.

Popup

Window placement
Center — popup appears centered on your screen. Tiles — each popup cascades offset from the previous one. Custom — remembers the position you last moved each service popup to.
Window size
Default — 1024×768. Large — 75% of the current window. Full screen — matches the current window size. Custom — starts at 1024×768 and remembers the size you resize each service popup to.

Menu

Show title
Show the "Google Account Chooser" heading at the top of the popup.
Show services
Show the quick-launch service buttons in the extension popup.
Show names below icons
Display service name labels below icons (or next to icons in list view).
Show options
Show the "Manage services & popup settings" link in the extension popup.
